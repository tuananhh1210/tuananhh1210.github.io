- là đơn vị mà n được tính dựa trên vậy lý: cm,mm,px,pt,inch em: đơn vị em này được so với kích thước font-size của đối tượng cha. VD: đối tương cha set font-size là 15px, thì đối tượng con set font-size là 1 em, nghĩa là font-size của con bằng 15px. rem: đơn vị rem này được so với kích thước font-size của trang html(thẻ html). Nếu html có font-size là 20px, thì 1 rem= 20px. Pseudo element
